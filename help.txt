==============================================================

--port, -p <port>         | Listen on <port>, defaults to 8999
--prefix, -P <prefix>     | Put all routes behind <prefix>
--registry, -r <registry> | Use <registry> for package data
--version, -v             | Print version information
--static, -s <dir>        | Override static files from <dir>
--title, -t <title>       | Set the title for your unpm-www
--help, -h                | This thing
